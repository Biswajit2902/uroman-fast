All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-03-06

### Added

- Initial release
- `romanize(text, lang=None)` — Convert text from any Unicode script to Latin characters
- Optional ISO 639-3 language code support for improved romanization
- Python 3.8+ support
- Rust-powered implementation via uroman-rs and PyO3
